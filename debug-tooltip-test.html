Observações do Layout:

  • Propriedades: Agora alinhadas à direita na primeira linha com inline styles para garantir funcionamento
  • Imagem: Permanece centralizada (32x32px)
  • Nome: Permanece centralizado em negrito
  • Tipo da Arma: Linha em itálico centralizada (do campo weaponType)
  • 🆕 Dano e Munição: Nova linha com sistema inteligente:
    • Melee: [⚔️] dano [calculado] - apenas dano
    • Ranged não-firearm: [⚔️] dano [calculado]    [●] munição_atual
    • Firearm: [⚔️] dano [calculado]    [●] atual/máximo
  • 🆕 Durabilidade: Nova linha com sistema de durabilidade:
    • Todas as armas: [🔨] atual/máximo (ex: 3/3)
    • Condicional: Só aparece se durability.max > 0
  • Descrição: Permanece centralizada em itálico
  • Layout Final: Propriedades → Imagem → Nome → Tipo → Dano/Munição → Durabilidade → Descrição
  • Solução: Usando inline styles para sobrescrever estilos do FoundryVTT

Sistema de Dano e Munição:

  • Ícone da Espada (fas fa-sword): Representa dano para todas as armas
  • Dano Base: Valor configurado no campo damage.value da arma
  • Dano Calculado [X]: Valor do damage.total (inclui força/destreza)
  • Ícone de Munição (fas fa-circle): Apenas para armas ranged com munição > 0
  • Munição Não-Firearm: Mostra apenas quantidade atual
  • Munição Firearm: Mostra atual/máximo (ex: 6/10)
  • Espaçamento: Dois espaços (  ) entre dano e munição

Sistema de Durabilidade:

  • Ícone da Forja (fas fa-hammer): Representa durabilidade da arma
  • Formato: atual/máximo (baseado em durability.current/durability.max)
  • Condicional: Só aparece se durability.max > 0
  • Posição: Entre Dano/Munição e Descrição
  • Exemplos: 3/3 (perfeita), 2/3 (levemente danificada), 1/3 (muito danificada)

Sistema de Artefatos Mágicos:

  • Formato: ◆ 🌀Artefato (símbolo de diamante + emoji de ciclone + texto)
  • Condicional: Só aparece se weapon.system.magicalArtifact === true
  • Posição: Embaixo da Descrição (última linha do tooltip)
  • Alinhamento: À esquerda (text-align: left)
  • Cor: #c9c7b8 (cor secundária padrão)
  • Exemplos: Espada Longa e Pistola Mágica são artefatos no teste

Sistema de Propriedades das Armas:

  • Formato: ◆ Propriedade1 ・ Propriedade2 ・ Propriedade3 (símbolo de diamante + separador middle dot)
  • Condicional: Só aparece se weapon.system.properties.length > 0 E houver propriedades válidas
  • Validação: Filtra propriedades vazias e garante que não apareça apenas o símbolo ◆
  • Capitalização: Primeira letra de cada propriedade em maiúscula, resto em minúscula
  • Posição: Embaixo do Artefato Mágico (se houver) ou embaixo da Descrição
  • Alinhamento: À esquerda (text-align: left)
  • Separador: ・ (middle dot / U+30FB) entre as propriedades
  • Cor: #c9c7b8 (cor secundária padrão)
  • Exemplos: Espada: "Afiado ・ Ambidestria ・ Duelar", Arco: "Preciso ・ Elegante", Pistola: "Explosiva ・ Rúnica ・ Encantada"
